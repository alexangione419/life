# Alex's Conway's Game of Life 

My implementation of Conway's Game of Life using C++'s SFML library. 

## Build Instructions

Requires CMake, a C++17 compiler, and SFML.

```bash
mkdir build && cd build
cmake ..
make
./bin/life
```

## Keyboard Controls 
- `Esc` - Exit the program
- `Spacebar` - Pause / Resume simulation 
- `R` - Reset the current iteration
- `N` - Generate a new random iteration
